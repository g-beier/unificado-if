CSS 01: Seletores e Regras

Vamos aprender como funcionam os seletores CSS e as regras para estilizar as páginas HTML.

O que é CSS

A linguagem CSS é a forma de estilizar as páginas HTML, garantindo que sua aparência seja responsiva e de acordo com o desejado no projeto.

Além de formatar cores, tamanhos e posições, é papel do CSS garantir que a página se adapte a todos os diferentes dispositivos que podem acessar a página: aparelhos como computadores, que acessam com telas de diferentes tamanhos; celulares, que são manipulados por toque, e cujo teclado altera o tamanho disponível da tela; dispositivos de acessibilidade, como leitores de tela; etc. Todas essas são preocupações que devemos ter para garantir a melhor UX.

Sintaxe

O CSS pode ser aplicado de três formas diferentes:

  • inline, através do atributo HTML style;
  • no arquivo HTML, através da tag HTML <style> no <head> do arquivo .html;
  • importando um arquivo .css externo, através de um <link>, também no <head> do arquivo .html

Nesta aula, vamos utilizar o segundo tipo: vamos editar o CSS deste arquivo HTML através da tag <style> que está no <head>.

Primeiro, abra o arquivo no editor de texto. Em seguida, encontre a tag <style> e delete o comentário /* CSS aqui */.

Dentro do <style>, onde estava o comentário, vamos inserir nossa primeira regra CSS: h1 {color: red;}.

  • h1 é o nosso seletor: é a parte da regra que determina quais são os elementos CSS afetados pelo estilo definido entre chaves.
  • color: red; é uma declaração CSS. As regras são formadas por diversas declarações. Repare que a declaração tem a separação com os dois pontos, e termina com ponto e vírgula.
    • color é a propriedade desta declaração; ela informa qual a propriedade que será alterada.
    • red é o valor que será assumido pela propriedade. Cada propriedade aceita valores diferentes. Neste caso, estamos alterando a cor do texto, através da propriedade color. Esta propriedade aceita cores em diferentes formados (HSL, RGB, HEX, etc.), além de um conjunto pré-definido de cores chave.

Podemos fazer com que esta regra se aplique a vários seletores, separando-os por vírgulas. Altere o seletor CSS para h1, h2, e o valor para darkred e observe o resultado.

Vamos alterar a cor do fundo (background-color) do corpo da nossa página (body) para o valor mistyrose. Tente fazer isso sozinho, e em caso de dúvida, clique no botão abaixo.

body { background-color: mistyrose; }

Cuide com a digitação, as chaves, dois pontos e ponto e vírgula no final

Nosso próximo passo será estilizar os elementos <code>, os pequenos pedaços de código ao longo do texto. Alguns deles se referem a código HTML, outros são pedaços de código CSS, e temos ainda extensões de arquivos. Todos eles já estão preparados para serem estilizados com classes.

Primeiro, estilize todos os <code> com as declarações display: inline-block; padding: 0.2em 0.5em; border-radius: 4px; . Estas são as declarações para deixar um espaço ao redor do elemento. O padding é aplicado ao redor do elemento, dentro da borda. Para isso ser aplicado em um elemento textual como o <code>, precisamos que ele tenha a propriedade display com o valor inline-block. Por fim, border-radius: 4px; deixa os cantos arredondados - o que será visível assim que atribuírmos cores aos elementos.

Cada um dos tipos diferentes serão declarados com um formato de cores diferentes, em regras diferentes. Estes formatos (RGB, HSL e HEX) permitem uma maior customização das cores, e são mais comuns em projetos reais. Para selecionarmos uma classe CSS, ao invés de apenas escrever o nome da classe, como fizemos com os elementos HTML, vamos iniciar com um ponto, como em .classSeguindo a tabela a seguir, monte as três regras CSS corretamente:

Por fim, vamos estilizar a tabela acima, de modo a facilitar a leitura. A primeira coisa, será observar que elementos de tabela no HTML são um tanto quanto complexos: eles são formados por diferentes tags, que desempenham papéis diversos. Na construção desta tabela, temos a seguinte estrutura:

  • table: a tabela como um elemento único.
    • tr : a linha de cabeçalho da tabela. Por vezes, é envolta em um elemento thead, especialmente se são várias linhas de cabeçalho. A tag tr significa table row, ou seja, linha da tabela.
      • Três th, table head. São as três células de cabeçalho da tabela, na primeira linha, que possuem a estilização em negrito. Note que a tag HTML é responsável pela diferença semântica, de sentido, desta célula em relação às outras. Enquanto cada célula traz um dado, um valor, esta diz a que este valor se refere: é um cabeçalho.
    • Três tr: a segunda, a terceira e a quarta linha da tabela.
      • Três td, table data, traduzidos para "datos da tabela". Cada td corresponde a uma das células da tabela, naquela linha. No caso, o primeiro é o Seletor, o segundo o background-color e o terceiro, a color.

Vamos aplicar as regras de forma progressiva, para compreender o papel de cada uma delas.

  1. A primeira coisa será centralizar a tabela, a partir dos valores de margem: table {margin: 2em auto;}. Ao declararmos dois valores para a propriedade margin, o primeiro valor será aplicado na vertical - espaçando a tabela dos parágrafos - e o segundo, na horizontal. O valor auto faz com que seja distribuído igualmente para ambos os lados, centralizando a tabela na página.
  2. Agora, vamos centralizar o texto da tabela: text-align: center;
  3. Vamos colocar uma borda que separe e facilite a leitura com a regra td { border: 1px solid black; }. Repare que esta regra se aplica apenas aos dados da tabela, e não ao cabeçalho. Para que todas as células tenham uma borda, adicione ao seletor o elemento th.
  4. As células da tabela não se encostam, todas tendo suas próprias bordas. Normalmente, as tabelas não são assim, não é mesmo? Podemos resolver isso com a declaração border-collapse: collapse; na regra do seletor table.
  5. Outra coisa muito importante para garantir a legibilidade é o whitespace, o espaço de respiro dos elementos. Para garantir isso, vamos adicionar um padding nas células da tabela (tanto nas <th> quanto <td>). Aplique o valor 0.2em 0.8em; na regra que já foi utilizada.
  6. Por fim, vamos colocar uma borda mais espessa, que diferencia a tabela dos demais elementos: coloque uma borda de 4px no elemento <table>, seguindo os exemplos anteriores.

Agora é com você

Estilize o botão que está na página, usando as regras que você aprendeu, com cores, bordas e espaçamentos diversos.
